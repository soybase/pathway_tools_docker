SoyCyc Soybean Metabolic Pathway Database

This the SoyCyc 10.0 Soybean Metabolic Pathway Database. The data in this database were inferred by ab-initio methods from the Glycine max Williams82 genome sequence gene calls (Wm82.a4.v1) from the Joint Genome Institute (JGI, Phytozome).

The "Search" button allows you to search for information in a number of ways.

Search ByDescription
CompoundsSearches using compound names.
Genes/Proteins/RNAsSearches using gene, protein or structural RNA names.
ReactionsSearches using biochemical reaction names.
PathwaysSearches using proper pathway names.
AdvancedPerform direct database queries.
OntologiesSearch for genes and pathway information using ontological terms from GO, EC, etc.

The "Tools" button allows you access to a number of computational tools to examine the database.


ToolDescription
Cellular OverviewPresents a schematic of known pathways and annotation data.
Omics ViewerAnnotate pathway information with data from transcriptomic experiments.
Comparative AnalysisIdentify pathways in common between the available databases.
ReportsGenerate descriptive reports for the database.
