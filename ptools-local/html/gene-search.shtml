Gene/Protein/RNA Search

Search by gene name or database identifier
Enter a gene name, or a database identifier from this database or from an external database to which this database contain links. Partial names will generate a substring search on gene names only (not on database identifiers).
Examples: "trpA", "trp", "b1236"
Search by protein or RNA name, EC number, or database identifier
Enter a protein or RNA name, an EC number, or a database identifier from this database or from an external database to which this database contain links. Partial names will generate a substring search on protein or RNA names only (not on database identifiers).
Examples: "tryptophan synthase", "P0A877", "1.1.2.3"

Search/Filter by evidence code
Select one or more evidence codes to filter the result to only include gene products that have the corresponding evidence for their function. Deselecting all evidence codes has the same effect as selecting all evidence codes -- no filtering will be done based on evidence code.
Search/Filter by cell component
Select one or more cell components to filter the result to only include gene products that have been assigned to those components. Note that not all proteins have been assigned to cell components. Deselecting all cell components has the same effect as selecting all cell components -- no filtering will be done based on cellular location.

Search/Filter by MultiFun term
Browse MultiFun to a particular gene, or select one or more MultiFun terms to filter your query to only retrieve genes annotated with that term or terms. Only terms that have genes annotated to them in this organism are available. If the MultiFun class above has no children, then this organism lacks any MultiFun annotations.
Search/Filter by organism
Type in or browse for an organism
Type in an organism name or NCBI Taxonomy ID, or browse the NCBI Taxonomy to select one or more organisms or organism classes. The results will be filtered to only include genes and products belonging those organisms. Organisms that are not present in the list have no gene products curated for them. Deselecting all organisms has the same effect as selecting all organisms -- no filtering will be done based on organism.

Search/Filter by existence of protein features
If multiple feature types are specified, return proteins that have of the specified types.

Select one or more protein feature types to filter the result to only include gene products for which features of those types have been identified. If no feature classes are listed, this database does not contain protein feature data.

Note: Only search criteria that have been checked at the time the query is submitted are included in the query. If multiple search criteria are specified, then results must satisfy ALL of them. Where applicable, a protein search will search all polypeptides, protein complexes, and their modified forms. For more search options, see the Advanced Search page.
