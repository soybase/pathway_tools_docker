Search/Filter by evidence code
Select one or more evidence codes to filter the result to only include pathways that are supported only by the corresponding evidence type. Deselecting all evidence codes has the same effect as selecting all evidence codes -- no filtering will be done based on evidence code.
Search/Filter by organism
Type in or browse for an organism
Type in an organism name or NCBI Taxonomy ID, or browse the NCBI Taxonomy to select one or more organisms or organism classes. The results will be filtered to include only pathways that are curated as occurring in those organisms (based on the experimental literature reviewed to date). Organisms that are not present in the list have no pathways curated for them. If a pathway has not been curated for a given organism, it may be truly absent from that organism, or our curators may not yet have reviewed the relevant literature. Deselecting all organisms has the same effect as selecting all organisms -- no filtering will be done based on organism.
Search/Filter by expected taxonomic range
Type in or browse for a taxon
Type in a taxon name or NCBI Taxonomy ID, or browse the NCBI Taxonomy to select one or more organisms or organism classes. The results will be filtered to include only pathways that are considered likely to occur in those organisms based on the estimated taxonomic range of each pathway. Taxonomic ranges are typically estimated by curators based on the experimentally observed range of a pathway. If an organism class cannot be further expanded, it means that the set of pathways does not change as you move further down in the organism taxonomy. Deselecting all organisms has the same effect as selecting all organisms -- no filtering will be done based on organism.

Include only engineered pathways
Include only chimeric pathways
Include only naturally occurring pathways
Do not filter based on engineered, chimeric or naturally occurring status
An engineered pathway is a pathway that has been constructed in a host organism by genetic modification or deletion of existing genetic traits and/or the introduction of genetic elements from other organisms. A chimeric pathway is a pathway comprised of reactions from multiple organisms that most commonly does not occur in its entirety in a single organism (Only sections of the pathway are likely to occur in their entirety in single organisms).
